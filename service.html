iRecとは・機能紹介

求職者から見つけてもらえる 採用サイト支援サービス

良い人と会うために大切なことは、 自社の魅力を発見し、「正しく」「魅力的に」「リアルタイムに」伝え続けること。 iRecは「伝える」「伝わる」採用サイトで、皆さんを精一杯サポートいたします。

の機能

Function 01 採用ページを作成する（サイト内容を設定・編集）
テーマカラー設定 自由に設定可能です。自社のコーポレートカラーに合わせて設定可能です。
サイト構成 最大で4つのカテゴリに分けてページを制作することができるので、「新卒」「中途」「アルバイト」など、見てほしいターゲットに合わせたメッセージや募集要項の出し分けが可能です。
ロゴ画像の登録 自社のロゴ画像の登録ができます。
コンテンツ作成 ［メッセージ］［事業内容］［社員インタビュー］［福利厚生］［キャリアパス］など、簡単に採用ページコンテンツを充実させることができます。
スマホ対応 自動的にスマホ表示に最適化されるので、スマホからの応募者にもデザインイメージを損なうこと無く対応できます。
メニュー管理 “メニューの表示・非表示を切り替えるだけで、新しいページを表示したり、公開ページを一時非公開にすることができます。また、ランディングページとしての利用も可能です。
フォーム一体型ページの作成 セミナー募集用のランディングページとしてご利用いただける、フォーム一体型のページをご用意しています。
Function 02 求人ページを作成する
求人原稿の項目 求人原稿ページは、企業様に合わせた項目が自由に設定できます。また画像やを入れることができ、仕事などの魅力を伝えるページ作成ができます。
ページ複製・求人コピー 作成済みのページを複製することができます。再入力やコピーアンドペーストの手間を削減できます。
CSV原稿投入 CSVで原稿ページ作成が可能です。更新・削除設定にも対応しています。
応募フォーム 応募フォームを設置可能です。フォームの項目や必須設定などは管理画面で簡単に設定できます。
Function 03 求人のアクセス数を増やす
大手求人検索サイト自動連携 「Indeed」「求人ボックス」「スタンバイ」「キャリアジェット」「Googleしごと検索」にも自動連携し、掲載することができます。より多くの人の目にとまる機会が増え、求人応募増加につながります。
SNS連携 採用サイトと各種SNS（Facebook、Twitter、Instagramなど）を連携させることができます。
バナー掲載 バナーを作成し、自社ホームページや外部サイトにリンクさせることができます。 バナー画像は自由に設定できます。
バナー掲載 ページタイトル（タグ）を任意で設定することが可能です。また、ページ単位でキーワード・descriptionの設定が可能です。ページ基本情報内のOGPを設定することで、主なSNS（facebookなど）との連携（ページのシェアなど）の際、ページの情報を正しく認識させることができます。
またfaviconの設置が可能です。
Function 04 採用サイトを分析する
タグ設置 GoogleAnalyticsやリスティングやIndeedプロモーションのコンバージョンタグも完了ページに簡単に設置することができます。
Function 05 採用サイトを分析する
ページ管理 各メニューに対して複数のページを作成でき、ページ単位で公開・非公開の設定が可能です。
ページの編集モード 公開済みのページは、閲覧しているそのページから編集画面を立ち上げ、編集することが可能です。
ページ公開日･公開終了日の設定 全てのページは、ページ公開期間を日単位で設定できます。
Function 06 セキュリティ
セキュリティ 全ページSSL対応
Function 07 ユーザー管理/ワークフロー
ユーザー管理 管理ページにログイン可能なユーザーを管理できます。ログインID、パスワードの設定や管理権限の設定が行えます。
編集権限設定 記事を編集する担当者、応募者管理をする担当者など、権限を割り当てることができます。
ワークフロー 担当者が作成したページはそのまま公開されることはなく、担当者と管理者との間で「申請」&「承認/却下」のやりとりが行われ、管理者が承認したページだけが公開されます。
